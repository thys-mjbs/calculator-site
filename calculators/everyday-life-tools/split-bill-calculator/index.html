About this split bill calculator

This split bill calculator helps you work out how much each person should pay when sharing a total bill. You enter the total bill amount, an optional tip or service charge percentage, and the number of people. The tool calculates the tip, the new total including tip, and the amount each person owes based on an equal split.

It is useful for restaurant outings, takeaways, shared taxis, group activities, and any other everyday situation where one bill needs to be shared fairly. You can enter whole numbers or decimals, and the inputs format with commas for easier reading while the calculator still uses the exact underlying values for the math.

Common questions

Do I have to include a tip or service charge?

No. The tip field is optional. If you want to split only the base bill, you can set the tip percentage to zero and the calculator will simply divide the bill by the number of people.

Can I use any currency or country?

Yes. The calculator is currency agnostic. It works with plain numbers only. You decide which currency the amounts represent and whether you are using it for local or international bills.

Does everyone have to pay the same amount?

This tool assumes an even split between all people. If some people should pay more or less, you can still use this as a baseline per person amount and then adjust manually based on what each person ordered or agreed to pay.
